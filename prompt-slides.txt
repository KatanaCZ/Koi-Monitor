CONTEXTE POUR L’IA
------------------
Crée une présentation slide (14 à 16 slides) sur **Koi Monitor** — application de monitoring système pour **Windows 10 et 11**, interface glassmorphisme thème **Koi Monitor Night**.

Public : utilisateurs lambda (gamers, streamers, créateurs, grand public curieux). Pas une présentation développeur.
Ton : professionnel, moderne, légèrement fun — esthétique **Koi & Sakura** (japonaise discrète, zen, néons cyberpunk).
Langue : français.
Priorités : (1) UI / modernité, (2) légèreté & perf au quotidien, (3) fiabilité des données, (4) utilité concrète.
Traduire le technique en bénéfice utilisateur (ex. « métriques GPU natives Windows » plutôt que « DXGI/WMI » sur la slide).

STRUCTURE DES SLIDES
--------------------

SLIDE 1 — Titre
Titre : Koi Monitor
Sous-titre : Surveillance système avec style.
Accroche : Application de monitoring pour Windows 10 & 11 — thème **Koi Monitor Night**, glassmorphisme Liquid Glass.
Phrase : Créé pour les amateurs de monitoring… et d’esthétique japonaise (Koi & Sakura).
Visuel : dashboard mockup, pétales sakura, dégradé violet → cyan

SLIDE 2 — La promesse
Titre : Votre PC, en un coup d’œil
Points :
- CPU, RAM, GPU, réseau — en direct et en couleur
- Connexion : meilleur DNS + « Prêt pour le jeu ? »
- Pilotes essentiels, sans fausses alertes
- Une app **belle à garder ouverte**, pas un utilitaire gris oublié

SLIDE 3 — Design Koi Monitor Night
Titre : Liquid Glass & Bento Paradigm
Points :
- Cartes Bento arrondies, effet **verre dépoli** (activable dans les paramètres)
- **Sans grain** sur le verre — rendu propre, pas granuleux
- Thème **sombre & clair** harmonisés (Daylight Neon en mode jour)
- Chaque composant a **sa couleur néon** — repérage instantané
Visuel : grille de widgets colorés, glass blur

SLIDE 4 — Identité & accueil
Titre : Une app qui a une signature
Points :
- **SlashTitle** : animation katana qui fend « Koi Monitor » au démarrage
- **Splash Screen** fluide : Initialisation → Matériel → DNS → Pilotes → Prêt
- Barre de chargement synchronisée au vrai chargement (pas de faux progrès)
- Barre de titre personnalisée — sans décorations Windows classiques
Visuel : splash + titre scindé

SLIDE 5 — StatsBar : le hub express
Titre : Liquid Glass Hub
Points :
- Anneaux néon : **CPU · RAM · GPU · Jeu · Actif**
- **Jeu** : badge verdict cliquable (manette)
- **Actif** : uptime compact (ex. `4h 32m` depuis le démarrage)
- Panneau latence jeu repliable sous la grille (dashboard)
Visuel : bandeau supérieur avec capsules néon

SLIDE 6 — Monitoring système
Titre : Votre machine en direct
Points :
- **CPU** 🩷 : usage, historique, mode *Spectrum Equalizer* (vue interactive par cœur)
- **RAM** 🔵 : utilisée / disponible + courbe
- **GPU** 🟣 : VRAM totale & allouée, charge 3D temps réel (métriques natives Windows)
- **Réseau** 🩵 : débit live ↓/↑, totaux cumulés, graphique ; **Wi‑Fi ou Ethernet** affiché

SLIDE 7 — DNS Monitor
Titre : Comparateur de serveurs DNS
Points :
- Teste Google, Cloudflare, Quad9, OpenDNS — **comparaison DNS**, pas latence jeu
- **Test auto** : 4 serveurs recommandés + lancement immédiat
- **Personnaliser** : choix des serveurs dans Paramètres → Réseau & DNS
- Verdict bandeau : *Excellente / Bonne / Moyenne / Critique* — meilleur serveur couronné ⚜️

SLIDE 8 — Latence jeu
Titre : « Prêt pour le jeu »
Points :
- Cellule **Jeu** dans le bandeau — verdict en temps réel
- Badges : *Prêt pour le jeu · Limite ranked · Wi‑Fi / box · …*
- Détail en un clic : **Passerelle** (box) · **Internet** · **Stabilité (jitter)**
- Mesure **indépendante du DNS** — pensée pour le multijoueur
- Fermeture : Fermer, re-clic badge, ou touche **Échap**
Visuel : badge vert / ambre / orange + panneau 3 colonnes

SLIDE 9 — Analyse des pilotes
Titre : Pilotes — l’essentiel, en toute honnêteté
Points :
- **Mode simplifié (défaut)** : 3 essentiels — GPU discret, réseau (Ethernet prioritaire), Bluetooth
- **Mode étendu** : + audio, stockage, iGPU, Wi‑Fi (scan ~1–2 min, avertissement à l’activation)
- Comparateur **version installée vs disponible** (segments diff surlignés)
- **Windows Update d’abord** — liens NVIDIA / AMD / Intel / Realtek en secours
- Statuts Safe UX : *Installé · Vérifier · MàJ dispo* — pas de fausse obsolescence
- Matériel **réel uniquement** — pas de bruit VM / VPN / périphériques inutiles

SLIDE 10 — Mode Zen (Méditation)
Titre : Mode Zen 🌸
Points :
- Bouton fleur de cerisier dans la barre de titre — bascule instantanée
- **Horloge centrale** + date FR + citation selon la charge du PC
- Télémétrie intégrée : CPU, RAM, GPU, Jeu, Actif (StatsBar masquée)
- **Antivirus réel** : *Protégé · [programme]* ou alerte
- États : Repos 💚 · Modéré 🩵 · Intense 💛
- **Économie de ressources** : graphiques lourds démontés ; sakura en faible/moyen

SLIDE 11 — Légèreté & empreinte
Titre : Légère pour une app moderne
Points :
- **~10–25 Mo** installé (bien en dessous des apps Electron ~150 Mo+)
- **~80–150 Mo** RAM au repos
- CPU **faible en veille** sur le dashboard
- Pics possibles uniquement au **scan pilotes** (mode étendu, ~1–2 min)
- Conçue pour tourner en fond pendant travail, stream ou jeu

SLIDE 12 — Fiabilité & confiance
Titre : Des données sérieuses, une UX rassurante
Points :
- Métriques lues **directement depuis Windows** — pas de chiffres inventés
- **DNS ≠ Latence jeu** — deux outils, deux questions, zéro confusion
- Pilotes : pas d’installation magique depuis l’app — **vous gardez le contrôle**
- Toasts clairs si DNS ou scan échoue — pas d’écran blanc
- Splash **non bloquant** (timeout sécurité ~90 s) — l’app démarre quand même
- Liens externes filtrés : sites constructeurs & Microsoft reconnus

SLIDE 13 — Accessibilité & ergonomie
Titre : Confortable pour tous
Points :
- Contrastes **WCAG AAA** — lisible en clair comme en sombre
- Navigation **clavier** complète
- **Échap** : ferme modales, panneau latence jeu, sort du Zen
- Réinitialisation paramètres : **double confirmation** (évite les erreurs)
- Notifications annoncées pour lecteurs d’écran (`aria-live`)

SLIDE 14 — Paramètres (personnalisation)
Titre : Ajustez Koi à votre rythme
Points :
- Intensité **Sakura** : Désactivé · Faible · Moyen · Élevé
- Intervalle métriques : 1 s · 2 s · 5 s · 10 s
- Intervalle DNS : 10 s · 15 s · 30 s · 1 min
- Serveurs DNS : checklist personnalisable
- **Glassmorphism** : activer / désactiver le flou vitré
- Mode pilotes simplifiés : activé par défaut

SLIDE 15 — Pour qui ?
Titre : Koi Monitor, c’est pour vous si…
Points :
- Vous jouez en ligne et voulez un verdict **Prêt pour le jeu**
- Vous stream / créez et suivez CPU & GPU sans dashboard moche
- Vous cherchez le **meilleur DNS** sans ligne de commande
- Vous voulez un **second écran zen** ou un monitoring honnête des pilotes
- Vous aimez les apps **légères, belles et fiables** sur Windows

SLIDE 16 — Clôture
Titre : Koi Monitor
Sous-titre : Surveillance système avec style.
Points :
- Koi Monitor Night · Liquid Glass · Sakura
- Windows 10 & 11 · 64 bits
- Open source — **MIT License** — Katana 2026
Phrase finale : *Créé avec passion pour le monitoring… et l’esthétique japonaise.*
Visuel : logo, sakura, néons de marque
